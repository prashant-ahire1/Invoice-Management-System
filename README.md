# Invoice Management System
<h2>📌 Project Overview</h2>

This is a full-stack Invoice Management System developed using React.js for the frontend and Java Spring Boot for the backend. The application helps users create, manage, update, and track invoices efficiently with a user-friendly interface.

<h3>🚀 Technologies Used</h3>
<h4>Frontend</h4>
<li>React.js</li>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
<li>Axios</li>
<li>Bootstrap / Tailwind CSS</li>

<h4>Backend</h4>
<li>Java</li>
<li>Spring Boot</li>
<li>Spring Data JPA</li>
<li>REST API</li>
<li>Hibernate</li>

<h4>Database</h4>
<li>MySQL</li>

<h4>✨ Features</h4>
<li>User Authentication & Login</li>
<li>Create Invoice</li>
<li>Update Invoice</li>
<li>Delete Invoice</li>
<li>Invoice Search & Filter</li>
<li>Customer Management</li>
<li>Product Management</li>
<li>Invoice PDF Generation</li>
<li>Responsive UI</li>
<li>RESTful API Integration</li>
